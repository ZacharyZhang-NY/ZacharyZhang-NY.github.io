What We Do

Defect Detection

The quality defect detection solution based on deep learning condenses a number of advanced technology applications in the field of computer vision and artificial intelligence, and incorporates a number of innovative detection concepts. Detection, while quickly detecting defects on the surface of materials, it can visually display the detection results, and the detection is accurate, stable and fast, which can greatly improve the efficiency of quality defect detection.

Pose Estimation

Pose estimation consists of locating various body parts (aka keypoints) that form a skeletal topology (aka links). Pose estimation has a variety of applications including gestures, AR/VR, HMI (human/machine interface), and posture/gait correction.

Object Detection

Object detection focuses on a specific object target, and requires to obtain the category information and location information of this target at the same time. Object detection gives an understanding of the foreground and background of the picture. We need to separate the target of interest from the background and determine the category and position of the target. Therefore, the output of the detection model is a list. Item uses a data set to give the class and location of the detected target.
